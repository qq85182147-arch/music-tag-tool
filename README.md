# 🎵 歌曲标签神器 (Music Tagger) v1.01
**歌曲标签神器 (Music Tagger)** 是一款专为本地音乐收藏家与发烧友打造的终极音频元数据批量编辑工具。
## ✨ 核心特性 (Features)
- 🎧 **全格式无损制霸**：完美支持 MP3, FLAC, M4A, WAV, 及顶级发烧原盘 DSF 格式。
- 🌐 **多源聚合搜索**：内置智能解析引擎，匹配网易云音乐、QQ音乐、酷狗音乐三大平台数据。
- 🖼️ **高阶元数据注入**：自动抓取并内嵌高清专辑封面与同步滚动歌词。
- 🤖 **无人值守挂机洗歌**：独创智能确认打分引擎，高分自动静默打签。
## 🚀 快速上手 (Quick Start)
本项目已通过 Nuitka 深度编译为底层 C 语言机器码，打造了免环境、极速响应的单文件独立运行版。前往 Releases 页面下载 歌曲标签神器v1.01.exe，双击即可运行。
